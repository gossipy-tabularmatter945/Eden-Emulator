# Eden Emulator

**Eden Emulator** is a high-accuracy emulator designed to run games from the Eden platform on modern PC hardware with improved performance, resolution scaling, and mod support.

## Key Features
- High compatibility with Eden games
- Support for high-resolution rendering and widescreen
- Save state and cheat support
- Mod compatibility (where applicable)
- Clean and user-friendly interface
- Regular updates for better performance and game support

## Getting Started
1. **Download** the latest version using the button above.
2. **Extract** the archive to any folder.
3. **Place your legal game files** in the correct directory.
4. **Run** `EdenEmulator.exe`.
5. Load your game and enjoy playing.

**Note:** You need legally obtained game files (not provided).

## System Requirements
- **Windows 10 / 11** or **Linux**
- Modern CPU (Intel i5 / AMD Ryzen 5 or better)
- GPU with Vulkan/OpenGL support
- 8 GB RAM (16 GB recommended)

## Security & Legal
**Important Notice:**  
You **must** own the original games and create your own dumps. Distributing or downloading copyrighted ROMs is illegal.

**Recommendations:**
- Download **only** from this official GitHub repository.
