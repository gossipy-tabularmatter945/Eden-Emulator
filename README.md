## ⚙️ Initial Configuration

The emulator requires specific files from a Nintendo Switch console to function. You must provide these files yourself to ensure legal compliance.

- **Prod Keys:** These files tell the emulator how to decrypt game data. Place your "prod.keys" file inside the "keys" folder generated by the emulator inside your local app data folder.
- **Firmware:** The system firmware provides the necessary interface for games to run. Open the emulator and go to the "File" menu, then select "Install Firmware." Choose the folder containing your system firmware files.
- **Game Files:** Place your game files (typically .nsp or .xci format) in a clear folder on your PC. Use the "Add Game Directory" option in the main user interface to tell the emulator where to look for your games.

## 🚀 Performance Optimization

If games run slowly or stutter, adjust these settings to improve your experience:

- **Graphics Backend:** Navigate to "Settings" then "Graphics." Switch between Vulkan and OpenGL. Vulkan usually provides better speed on most modern systems.
- **Resolution:** Lower the resolution scale to 0.75x or 1x if you encounter frame rate drops.
- **V-Sync:** Enable or disable V-Sync to prevent screen tearing or input lag.
- **Cache:** Allow the emulator to build its shader cache. You might notice small stutters during the first few minutes of gameplay as the emulator learns how to display assets. These stutters disappear once the cache completes.

## 🔧 Troubleshooting Common Issues

Check this list if you encounter errors during startup or gameplay:

- **Missing Files:** The application will trigger an error if the "prod.keys" file is missing or corrupted. Verify that your keys match the current firmware version.
- **Application Crash:** Update your Windows display drivers. Outdated drivers cause most stability issues.
- **Black Screen:** Ensure that you installed the system firmware. The emulator cannot launch software without these core files.
- **Input Lag:** Connect your controller through a wired USB connection rather than Bluetooth to reduce input delay.
- **Audio Stutter:** Increase the audio output latency in the settings menu if you hear crackling or choppy sound.
